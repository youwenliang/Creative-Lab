Firefox OS Projects

Firefox OS - Smart TV Prototype

Firefox OS Smart TV is one of our main products in the Taipei office. While we were developing the project, we wanted to test out some new features so I made a high-fidelity prototype implemented with the features we wanted to test. We've used this prototype to let multiple participants to navigate through and try out new functions using actual remote control (This web version prototype can be control by keyboard.)

http://people.mozilla.org/~mliang/FirefoxOS_TV-Prototype/

View project on GitHub

Firefox OS - Add-On Prototype

This Add-on prototype is one of our early concepts about adding a universal button on top of our Firefox OS interface. It acts as a floating menu button with the most used function in it, such as "Home", "Back", "Scroll Top" and "Task Manager". I used Firefox Web IDE tools to develop this add-on using basic Javascript and CSS.

http://people.mozilla.org/~mliang/FirefoxOS_AddOn-Prototype/Add-On-Video.mp4

View project on GitHub

Firefox OS - Alopex Concept Prototype

Alopex is a Firefox OS concept from the UX team, using mobile web browsering experience as its core and build an OS around that. Basically creating a truely web-based operating system. I was practicing FramerJS and made this interactive prototype to learn more about FramerJS.

http://people.mozilla.org/~mliang/FirefoxOS_Alopex-Prototype.framer/

View project on GitHub

Firefox OS - FxOS Cloud Intro Animation

FxOS Cloud is a concept proposal for our data sync team. We want to provide a solution for Firefox OS users to sync their files and be able to share them easily. We also thought about task continuity, creating a seamless experience from devices to devices. Here's a video I made as a pitch for the concept.

http://people.mozilla.org/~mliang/FirefoxOS_Cloud/FirefoxOS_Cloud.mp4

Connected Devices - Project Webby Intro Animation
